=========

根据fis以及rsd构建自己的项目目录结构
###第一步：安装工具
rsd是基于fis而包装的简单命令行工具，而fis的安装需要[nodejs](http://nodejs.org/)环境的支持
同时需要你熟悉一下[fis](http://fis.baidu.com)
使用nodejs的包管理器node来安装
```bash
npm install -g rsd
```
